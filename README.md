# 🧬 Breast Cancer Prediction App

This project predicts whether a breast tumor is **Benign** or **Malignant** using a hybrid machine learning model.

The model combines **XGBoost, LightGBM, and Logistic Regression** and is deployed as an interactive web application using **Streamlit**.

## Tech Stack

* Python
* Scikit-learn
* XGBoost
* LightGBM
* Streamlit
* Pandas & NumPy

## Run the App

Install dependencies:

```
pip install -r requirements.txt
```

Run the Streamlit app:

```
streamlit run app.py
```

The app will open in your browser.
